🧩 1. Group, Artifact, and Package Name

✅ Group (like a company or domain)

  • If you own mycompany.com → com.mycompany

  • Personal project → com.johnsmith or io.github.johnsmith

👉 This ensures uniqueness across the Java ecosystem.

✅ Artifact (your project name)

  • This is the actual application name

  • Lowercase, no spaces (use hyphens if needed)

Format:

project-name

Examples:

  • inventory-service

  • user-api

  • demo

✅ Package Name (auto-generated)

Spring Initializr combines them:

group + artifact

Example:

  • Group: com.mycompany

  • Artifact: inventory-service

➡️ Package:

com.mycompany.inventoryservice

👉 Notice: hyphens are removed in package names.

🔑 Simple Rule of Thumb

FieldWhat it representsExample
GroupOrganization/domaincom.mycompany
ArtifactProject/app nameorder-service
PackageJava namespacecom.mycompany.orderservice

📦 2. Jar vs War — Which Should You Choose?

This is about how your app runs.

✅ JAR (Recommended for most cases)

✔ Default choice for Spring Boot
✔ Runs as a standalone app
✔ Has embedded server (like Apache Tomcat)

How it runs:

Bash
java -jar app.jar

Use JAR if:

  • You’re building REST APIs

  • Microservices

  • Modern cloud apps (Docker, Kubernetes)

  • You want simplicity

👉 90% of projects → choose JAR

⚠️ WAR (older/traditional deployment)

✔ Used for deploying to external servers like:

  • Apache Tomcat (standalone)

  • JBoss EAP

  • WebLogic Server

Use WAR if:

  • Your company requires deployment to a shared app server

  • You’re working with legacy systems

⚔️ JAR vs WAR Summary

FeatureJAR ✅ (modern)WAR ⚠️ (legacy use)
DeploymentStandaloneExternal server
SetupSimpleMore complex
Use caseMicroservices/APIsEnterprise legacy apps
Recommendation⭐ Default choiceOnly if required

For most beginners and modern development:

  • Package → auto-generated

  • Packaging → ✅ JAR

FieldWhat it does

ValueBehavior

package com.rohith.javakafka.service;

import com.rohith.javakafka.model.User;
import com.rohith.javakafka.repository.UserRepository;
import org.springframework.beans.factory.annotation.Autowired;
import org.springframework.stereotype.Service;

import java.util.List;
import java.util.Optional;

@Service
public class UserService {

    @Autowired
    private UserRepository userRepository;

    // Create
    public User createUser(User user) {
        return userRepository.save(user);
    }

    // Read all
    public List<User> getAllUsers() {
        return userRepository.findAll();
    }

    // Read by ID
    public Optional<User> getUserById(Long id) {
        return userRepository.findById(id);
    }

    // Update
    public User updateUser(Long id, User updatedUser) {
        return userRepository.findById(id).map(user -> {
            user.setName(updatedUser.getName());
            user.setEmail(updatedUser.getEmail());
            return userRepository.save(user);
        }).orElseThrow(() -> new RuntimeException("User not found with id " + id));
    }

    // Delete
    public void deleteUser(Long id) {
        userRepository.deleteById(id);
    }

    // Custom query
    public List<User> findUsersByName(String name) {
        return userRepository.findByNameContains(name);
    }
}

package com.rohith.javakafka.controller;

import com.rohith.javakafka.model.User;
import com.rohith.javakafka.service.UserService;
import org.springframework.beans.factory.annotation.Autowired;
import org.springframework.web.bind.annotation.*;

import java.util.List;

@RestController
@RequestMapping("/api/users")
public class UserController {

    @Autowired
    private UserService userService;

    // Create
    @PostMapping
    public User createUser(@RequestBody User user) {
        return userService.createUser(user);
    }

    // Read all
    @GetMapping
    public List<User> getAllUsers() {
        return userService.getAllUsers();
    }

    // Read by ID
    @GetMapping("/{id}")
    public User getUserById(@PathVariable Long id) {
        return userService.getUserById(id)
                .orElseThrow(() -> new RuntimeException("User not found"));
    }

    // Update
    @PutMapping("/{id}")
    public User updateUser(@PathVariable Long id, @RequestBody User user) {
        return userService.updateUser(id, user);
    }

    // Delete
    @DeleteMapping("/{id}")
    public void deleteUser(@PathVariable Long id) {
        userService.deleteUser(id);
    }

    // Custom query: find users by name substring
    @GetMapping("/search")
    public List<User> searchUsers(@RequestParam String name) {
        return userService.findUsersByName(name);
    }
}
